## postgresql12-contrib 패키지
- PostgreSQL에서 기본적으로 제공하는 확장(Extensions)을 제공하는 패키지.

#### 설치방법
```
yum -y install postgresql2-contrib
```

#### 확장 설치방법
```sql
create extension extension_name;
```
