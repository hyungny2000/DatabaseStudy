# GiST 인덱스

## 참고자료

## 정의
- Generalized Search Tree의 약자로, Banalced Tree라는 점에서 B-Tree 인덱스와 유사함
- B-Tree인덱스와 달리 비정형 데이터 (공간데이터, 텍스트 유사도, K-근접 검색) 검색에 다양하게 사용됨.
- 데이터 타입에 따라 다양한 확장을 설치하여 활용함
