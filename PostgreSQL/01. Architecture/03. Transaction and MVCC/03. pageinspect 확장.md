## pageinspect 확장

#### 용도
- 매우 낮은 수준의 페이지 정보를 확인하는 용도

#### 설치
```
create extension pageinspect
```

#### 기능
###### 페이지 헤더 정보 조회
```
SELECT * FROM page_header(get_raw_page('pg_class', 0));
```
- 테이블명, 페이지 번호로 조회

###### 페이지에 있는 튜플 정보 조회
```
SELECT * FROM bt_page_items('pg_class', 0);
```
- 테이블명, 페이지 번호로 조회
